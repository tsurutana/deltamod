DeltaMod
====

DeltaMod is a 3D modeling software for deltahedron, allows you to keep equilateral triangle faces during editing.

## Features
* Augmentation
* (Gyro)elongation
* Tucking
* Subdivision
* Regularization

## Requirement
Java 1.8 or later.

## Installation
To install DeltaMod, download as a zip and unzip it, then, run /build/libs/DeltaMod_1.0.jar.
